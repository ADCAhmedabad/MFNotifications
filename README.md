# MFNotifications
Demonstrating iOS 10 Rich Notifications with Images, Video, Gifs.
--

![](./Notification1.gif)

OS Version 
--
- iOS 10 (Some of the notifications will work on iOS 9)

How to run
--
- Update Xcode to at least 8.0
- Open project xcodeproj
- In Xcode press Cmd-R or choose Run
